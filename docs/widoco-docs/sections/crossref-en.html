Comprehension Awarenessc back to ToC or Class ToC

IRI: https://www.w3id.org/rcao#ComprehensionAwareness

Level 2: Represents a synthetised understanding of the meaning or significance of perceived information. An instance of this class is typically derived from one or more PerceptualAwareness states. For example, from perceiving a vehicle's position and its brake light status, a Level 2 state can be inferred: 'Subject S is aware that the vehicle ahead is braking hard, constituting a potential hazard' [Source 371].
